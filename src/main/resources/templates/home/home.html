API END POİNTLERİ

http://localhost:8080 yazdıktan sonra ;

Bütün ilçeler: /districts

ID'ye göre getir: /districts/(ID yazın)

Kıtaya göre getir: /districtsByContinent?continent=(Kıta adı yazın)

Nüfusa göre getir: /districtsByDescPopulation?population=(Nüfusu buraya yazın)

İlçe adına göre getir: /districtsByName?name=(İlçe adını yazın)

Delete,Post,Put istekleri için Postman vb. programla yapılacaklar:

ID'ye göre sil: /districts/(ID yazın,Postman'da delete isteği yapınız)

Yeni ilçe oluştur: /districts(Postman'da post isteği yapınız)

İlçeyi güncelle: /districts/(ID yazın,Postman'da put yapınız)

DTO örneği için Browser veya Postman'la kıta ve nüfusa göre işlemler:

Browser'la Örnek: /districtsDtoBrowser?continent=AVRUPA&population=200000

Postman'la Örnek: istek gövdesine "kıta"="ANADOLU","nüfus"=300000 yazıp,

/districtsDto'ya get isteği gönderin
